# QgRuns

큭런은 여러 프로그램을 한번에 대신 실행할 때 쓰려고 만들었어요

## 옵션
|옵션|설명|
|------|---------|
|-interval=[밀리초]|여러 프로그램 실행할 때 사이 사이 간격. 밀리초 만큼 쉬었다가 실행해요|
|-test|실행은 안하고 메시지 박스로 그냥 보여줘요|
|-nocheck|실행할때 파일이 이미 실행 중인지 검사 안해요|
|-silence|메시지 박스가 나올일이 있더라도 안보여줘요

## 명령
|명령|설명|
|------|---------|
|RUN|실행해요. 이 명령은 생략할 수 있어요|
|COPY|복사해요 COPY (원본) (대상) [[옵션]] 으로 쓸 수 있고, 대상은 디렉터리가 아닌 파일 이름으로 지정해야 해요|
|MOVE|옮겨요. COPY랑 사용방법은 똑같아요|
